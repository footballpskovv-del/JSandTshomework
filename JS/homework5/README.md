# SWAPI Planets App

Учебный проект по JavaScript.

## Функционал

* Загрузка планет из SWAPI
* Пагинация по 10 элементов
* Просмотр информации о планете
* Модальное окно
* Загрузка фильмов планеты
* Загрузка персонажей планеты
* Отображение родного мира персонажей
* Кнопка "Показать все"

## Использованные технологии

* HTML
* CSS
* JavaScript
* Fetch API
* Promise
* Promise.all
* Async/Await

## API

SWAPI:
https://swapi.py4e.com/

## Запуск

Открыть файл `index.html` через Live Server.
